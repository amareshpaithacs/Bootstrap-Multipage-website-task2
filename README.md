# ITCO Website Clone

Frontend clone of the ITCO website using HTML, CSS, and Bootstrap 5.

## Pages Included

* Home
* About
* Services
* Team
* Blog Details
* Contact

## Technologies Used

* HTML5
* CSS3
* Bootstrap 5

## Features

* Fully Responsive Design
* Bootstrap Navbar & Dropdown
* Smooth Hover Effects
* Animations & Transitions
* Mobile Friendly Layout
* Reusable Components

## How to Run

1. Download the project
2. Open the ITCO folder
3. Run `index.html` using Live Server
